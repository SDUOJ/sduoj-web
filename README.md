# sduoj-web

## 配置环境

```bash
git clone https://github.com/SDUOJ/sduoj-web.git
cd sduoj-web
npm install
```

### Compiles and hot-reloads for development

```bash
npm run serve
```

### Compiles and minifies for production

```bash
npm run build
```

### Lints and fixes files

```bash
npm run lint
```
